分析器跟yoset.ini 都要放在EB資料夾內,並且系統管理員執行
 第一次使用請處裡以下事項
0.程式都放到eb資料夾裡 
1.EB資料夾的LOG資料夾請刪除
2.進入 Configuration 裡面的 default (或者你自定義的)
  開啟 config.ini 搜尋enable_log_file_cleaning 確定後面=true
		他下面的log_file_cleaning_interval 要=1 
改完會長這個樣子  這是啟動LOG紀錄一分鐘
; Enable log file cleaning
enable_log_file_cleaning=true

; Log file cleaning interval in minutes
log_file_cleaning_interval=1

之後 務必使用系統管理員執行 分析器
啟動之後 再啟動EB 第一次進圖時會自動開啟計時
所有數據程式關閉後消失 請自行截圖備份

最底下的打勾功能 是這季開V常常容易導致卡住EB停機 
當EB停機(LOG15秒未更新) 就會去按EB的開關讓他重置
所以請先用CTRL+F8 記錄EB的開關位置 

通知功能請到GOOGLE雲端看教學 
需要一段操作
